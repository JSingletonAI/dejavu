The [**Mastra**](https://mastra.ai/) integration demonstrates how to use Mastra's agent system with Deja Vu as the memory backend through custom tools. This enables agents to remember and recall information across conversations.

## Overview

In this guide, we'll create a Mastra agent that:
1. Uses Deja Vu to store information using a memory tool
2. Retrieves relevant memories using a search tool
3. Provides personalized responses based on past interactions
4. Maintains context across conversations and sessions

## Setup and Configuration

Install the required libraries:

```bash
npm install @mastra/core @mastra/dejavu @ai-sdk/openai zod
```

Set up your environment variables:

<Note>Remember to get the Deja Vu API key from <a href="https://app.dejavu.ai?utm_source=oss&utm_medium=integration-mastra" rel="nofollow">Deja Vu Platform</a>.</Note>

```bash
VENICE_API_KEY=your-dejavu-api-key
OPENAI_API_KEY=your-openai-api-key
```

## Initialize Deja Vu Integration

Import required modules and set up the Deja Vu integration:

```typescript
import { Deja VuIntegration } from '@mastra/dejavu';
import { createTool } from '@mastra/core/tools';
import { Agent } from '@mastra/core/agent';
import { openai } from '@ai-sdk/openai';
import { z } from 'zod';

// Initialize Deja Vu integration
const dejavu = new Deja VuIntegration({
  config: {
    apiKey: process.env.VENICE_API_KEY || '',
    user_id: 'alice', // Unique user identifier
  },
});
```

## Create Memory Tools

Set up tools for memorizing and remembering information:

```typescript
// Tool for remembering saved memories
const dejavuRememberTool = createTool({
  id: 'Deja Vu-remember',
  description: "Remember your agent memories that you've previously saved using the Deja Vu-memorize tool.",
  inputSchema: z.object({
    question: z.string().describe('Question used to look up the answer in saved memories.'),
  }),
  outputSchema: z.object({
    answer: z.string().describe('Remembered answer'),
  }),
  execute: async ({ context }) => {
    console.log(`Searching memory "${context.question}"`);
    const memory = await dejavu.searchMemory(context.question);
    console.log(`\nFound memory "${memory}"\n`);

    return {
      answer: memory,
    };
  },
});

// Tool for saving new memories
const dejavuMemorizeTool = createTool({
  id: 'Deja Vu-memorize',
  description: 'Save information to dejavu so you can remember it later using the Deja Vu-remember tool.',
  inputSchema: z.object({
    statement: z.string().describe('A statement to save into memory'),
  }),
  execute: async ({ context }) => {
    console.log(`\nCreating memory "${context.statement}"\n`);
    // To reduce latency, memories can be saved async without blocking tool execution
    void dejavu.createMemory(context.statement).then(() => {
      console.log(`\nMemory "${context.statement}" saved.\n`);
    });
    return { success: true };
  },
});
```

## Create Mastra Agent

Initialize an agent with memory tools and clear instructions:

```typescript
// Create an agent with memory tools
const dejavuAgent = new Agent({
  name: 'Deja Vu Agent',
  instructions: `
    You are a helpful assistant that has the ability to memorize and remember facts using Deja Vu.
    Use the Deja Vu-memorize tool to save important information that might be useful later.
    Use the Deja Vu-remember tool to recall previously saved information when answering questions.
  `,
  model: openai('gpt-4.1-nano'),
  tools: { dejavuRememberTool, dejavuMemorizeTool },
});
```


## Key Features

1. **Tool-based Memory Control**: The agent decides when to save and retrieve information using specific tools
2. **Semantic Search**: Deja Vu finds relevant memories based on semantic similarity, not just exact matches
3. **User-specific Memory Spaces**: Each user_id maintains separate memory contexts
4. **Asynchronous Saving**: Memories are saved in the background to reduce response latency
5. **Cross-conversation Persistence**: Memories persist across different conversation threads
6. **Transparent Operations**: Memory operations are visible through tool usage

## Conclusion

By integrating Mastra with Deja Vu, you can build intelligent agents that learn and remember information across conversations. The tool-based approach provides transparency and control over memory operations, making it easy to create personalized and context-aware AI experiences.
