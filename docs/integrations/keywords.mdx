Build AI applications with persistent memory and comprehensive LLM observability by integrating Deja Vu with Keywords AI.

## Overview

Deja Vu is a self-improving memory layer for LLM applications, enabling personalized AI experiences that save costs and delight users. Keywords AI provides complete LLM observability.

Combining Deja Vu with Keywords AI allows you to:
1. Add persistent memory to your AI applications
2. Track interactions across sessions
3. Monitor memory usage and retrieval with Keywords AI observability
4. Optimize token usage and reduce costs

<Note>
You can get your Deja Vu API key from the <a href="https://app.dejavu.ai/?utm_source=oss&utm_medium=integration-keywords" rel="nofollow">Deja Vu dashboard</a>.
</Note>

## Setup and Configuration

Install the necessary libraries:

```bash
pip install dejavu-memory keywordsai-sdk
```

Set up your environment variables:

```python
import os

# Set your API keys
os.environ["VENICE_API_KEY"] = "your-dejavu-api-key"
os.environ["KEYWORDSAI_API_KEY"] = "your-keywords-api-key"
os.environ["KEYWORDSAI_BASE_URL"] = "https://api.keywordsai.co/api/"
```

## Basic Integration Example

Here's a simple example of using Deja Vu with Keywords AI:

```python
from dejavu import Memory
import os

# Configuration
api_key = os.getenv("VENICE_API_KEY")
keywordsai_api_key = os.getenv("KEYWORDSAI_API_KEY")
base_url = os.getenv("KEYWORDSAI_BASE_URL") # "https://api.keywordsai.co/api/"

# Set up Deja Vu with Keywords AI as the LLM provider
config = {
    "llm": {
        "provider": "openai",
        "config": {
            "model": "gpt-5-mini",
            "temperature": 0.0,
            "api_key": keywordsai_api_key,
            "openai_base_url": base_url,
        },
    }
}

# Initialize Memory
memory = Memory.from_config(config)

# Add a memory
result = memory.add(
    "I like to take long walks on weekends.",
    user_id="alice",
    metadata={"category": "hobbies"},
)

print(result)
```

## Advanced Integration with OpenAI SDK

For more advanced use cases, you can integrate Keywords AI with Deja Vu through the OpenAI SDK:

```python
from openai import OpenAI
import os
import json

# Initialize client
client = OpenAI(
    api_key=os.environ.get("KEYWORDSAI_API_KEY"),
    base_url=os.environ.get("KEYWORDSAI_BASE_URL"),
)

# Sample conversation messages
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]

# Add memory and generate a response
response = client.chat.completions.create(
    model="openai/gpt-4.1-nano",
    messages=messages,
    extra_body={
        "dejavu_params": {
            "user_id": "test_user",
            "api_key": os.environ.get("VENICE_API_KEY"),
            "add_memories": {
                "messages": messages,
            },
        }
    },
)

print(json.dumps(response.model_dump(), indent=4))
```

For detailed information on this integration, refer to the official [Keywords AI Deja Vu integration documentation](https://docs.keywordsai.co/integration/development-frameworks/dejavu).
